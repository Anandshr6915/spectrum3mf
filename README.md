## 🧪 Palette mapping

Palette mapping helps you match model colors to the colors available for your print.

You can use it to:

- assign colors to a fixed set of filament colors
- keep the print closer to the source texture
- reduce color changes in the final file
- simplify busy textures into cleaner print layers

A good workflow is:

1. Start with the main colors in your model.
2. Pick the closest palette colors.
3. Check the preview.
4. Adjust until the result looks right.
5. Export the file when the match looks good.

## 🧩 Virtual mixes

Virtual mixes let you define mixed colors that do not exist as direct palette entries. This is useful when you want a better match between the source texture and the print palette.

Use virtual mixes when:

- the model has skin tones, gradients, or soft shading
- the base palette feels too limited
- you want a closer visual match
- you need more control over the final look

A few simple mix ideas:

- red + white for soft pink tones
- blue + white for light blue shades
- yellow + red for warm orange tones
- black + white for grays

## 🖨️ Best use with Snapmaker U1

This app targets multicolor 3MF files for Snapmaker U1. For best results:

- use a clean palette
- keep the number of colors low when possible

## 🧼 Troubleshooting color results

If the colors do not look right:

- try a different palette
- reduce the number of virtual mixes
- use stronger contrast in the source image
- check that the texture file matches the model
- test with a simpler model first

If the exported 3MF looks flat, the source texture may use too many near-matching shades. A tighter palette can help.
