# Code of Conduct

## Our Standards

Participants are expected to:

- Be respectful and constructive.
- Focus feedback on code and behavior, not people.
- Assume good intent and ask clarifying questions.
- Avoid harassment, hate speech, and personal attacks.

## Enforcement

Project maintainers may remove comments, close discussions, or block participation for behavior that violates this code.

## Scope

This applies to issues, pull requests, discussions, and other project communication channels.
