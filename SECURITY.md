# Security Policy

## Supported Versions

This project is early-stage. Security fixes are applied on the default branch.

## Reporting a Vulnerability

- Use GitHub's private vulnerability reporting for this repository.
- Do not publish exploit details in public issues.
- Include reproduction steps, impact, and affected files/paths.

## Response Goals

- Initial triage within 7 days.
- A fix or mitigation plan as soon as practical, based on severity.
